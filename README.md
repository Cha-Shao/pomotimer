## 命令

```sh
# install dependencies
$ pnpm install

# serve with hot reload at localhost:3016
$ pnpm dev

# generate the final static site
$ pnpm build
```

## 环境变量
`.env.local`

#### HIGHTLIGHT_PROJECT_ID

(可选) 是否使用highlight，如果是，填入projectId
